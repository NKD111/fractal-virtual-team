## Stack

- **Backend:** Node.js 18+, Express, Socket.io, BullMQ
- **AI:** Claude API, LangChain, Pinecone, ElevenLabs
- **DB:** Supabase (PostgreSQL + pgvector), Redis
- **Frontend:** Next.js 14, TypeScript, PixiJS, Tailwind
- **Deploy:** Railway (backend), Vercel (frontend)

## Quick Start

```bash
cd backend && npm install && npm run dev
cd frontend && npm install && npm run dev
```
