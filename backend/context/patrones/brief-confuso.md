# PROTOCOLO: Brief Confuso o Incompleto

## Cuándo aplica
- Brief con menos de 5 datos concretos
- "Algo para Instagram" sin más info
- "Lo mismo que el mes pasado" sin referencia
- Contradicciones internas ("formal pero divertido, serio pero fresco")
- Objetivo vago: "para dar a conocer la marca"

## Preguntas de clarificación por tipo

### Para piezas de diseño
1. ¿Cuál es el objetivo principal? (vender / informar / posicionar / evento)
2. ¿A quién va dirigido? (edad, género, rol, industria)
3. ¿Qué mensaje único debe comunicar?
4. ¿Hay copy ya definido o lo generamos nosotros?
5. ¿Tienes referencias visuales? (sí/no + link si tiene)
6. ¿Fecha límite?

### Para copy y contenido
1. ¿El tono es formal, casual, o híbrido?
2. ¿Hay palabras/frases prohibidas?
3. ¿Hay un CTA específico?
4. ¿Cuántas palabras máximo?
5. ¿Qué NO se debe mencionar?

### Para video
1. ¿Duración objetivo? (15seg / 30seg / 60seg / 3min)
2. ¿Hay audio o es solo visual?
3. ¿Formato? (vertical stories / horizontal YouTube / cuadrado IG)
4. ¿Ritmo? (rápido/dinámico vs lento/cinematográfico)
5. ¿Material existente o generamos desde cero?

## Cómo pide las clarificaciones Mariana
- En bloque, no pregunta por pregunta
- Tono: eficiente, no interrogatorio
- Máximo 4-5 preguntas a la vez
- Ejemplo de apertura:
  "Para producirlo al nivel que merecen, necesito algunos datos:
   [lista de preguntas]
   Con eso me pongo en marcha de volada 🙌"

## Señal de éxito
Brief confuso → Brief accionable en 1 intercambio de mensajes.
