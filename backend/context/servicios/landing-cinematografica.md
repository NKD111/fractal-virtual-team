# SERVICIO: Landing Cinematográfica

## Descripción
Landing page de alta conversión en Next.js.
Diseño editorial, copy persuasivo, optimización SEO desde el inicio.
El nombre "cinematográfica" refleja el nivel de producción y dirección de arte.

## Precios
- Estándar (10 secciones): $1,500 USD
- Premium (10+ secciones + video background): $3,000 USD
- Premium + Parrilla: $3,000 USD setup + $800/mes

## Qué incluye
### Estándar $1,500 USD
- 10 secciones: Hero → Problema → Solución → Para quién → Proceso → Credibilidad → Testimonios → Precios → FAQ → CTA Final
- Copy completo en español mexicano
- Diseño editorial premium (Next.js + Tailwind)
- SEO optimizado desde el inicio (title, meta, H1, schema)
- WhatsApp flotante + GA4 + Meta Pixel configurados
- Responsive móvil-primero
- Entrega: 7-10 días hábiles

### Premium $3,000 USD
- Todo lo de Estándar más:
- Video background en hero (loop 10-15 seg)
- Secciones avanzadas según brief
- Formulario de captura con integración
- Entrega: 10-15 días hábiles

## Proceso
1. Brief estructurado (cuestionario-landing.md)
2. Propuesta de estructura + copy (ALEX)
3. Moodboard visual (CARLOS)
4. Desarrollo en Next.js (ATLAS)
5. Revisión interna (VALENTINA + QC-BOT)
6. Aprobación NKD
7. Entrega + instrucciones de deploy

## Stack tecnológico
- Next.js 14 + App Router
- Tailwind CSS
- Shadcn/UI (componentes)
- Vercel (deploy recomendado)
- Analytics: GA4 + Meta Pixel
