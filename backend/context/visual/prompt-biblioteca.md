## Estructura de un prompt completo

```
[ESTILO VISUAL BASE]
[SUJETO / ESCENA PRINCIPAL]
[COMPOSICIÓN / ENCUADRE]
[ILUMINACIÓN]
[DETALLES TÉCNICOS]
[NEGATIVO — qué NO incluir]
```

```
Premium editorial-commercial franchise expo design.

Left side: structured information modules with navy blue blocks.
Right side: large curved photo window showing Mexican franchise expo hall.
Professional entrepreneurs networking. Warm lighting.
Strong typographic hierarchy. Gotham-style bold titles.

Brand colors: Navy #0B2A4A, Red #C8102E, White #FFFFFF.

NEVER: text overlay on faces, fake logos, messy composition, neon colors.
```
